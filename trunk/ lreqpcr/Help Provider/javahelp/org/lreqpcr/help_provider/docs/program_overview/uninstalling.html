Uninstalling the LRE Analyzer

Go to the directory in which the LRE Analyzer was installed (e.g. "C:/Program Files/lreanalyzerXXX" which is the default for Windows), which should contain the following files and directories:

Delete the Settings.lre file, or if you wish to keep your settings, move the Settings.lre file to a temporary location. Start the uninstall.exe program and follow the instructions. After installing a new version, move the Settings.lre file back to the installation directory, or, if it is absent a new Settings.lre file will created when the program is first opened.
