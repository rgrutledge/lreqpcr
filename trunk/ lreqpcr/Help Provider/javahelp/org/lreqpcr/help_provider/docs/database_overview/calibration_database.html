Calibration Database

Similar to the Experiment Database, data is stored in a tree-like structure in which calibration profiles are listed under the corresponding run. An "Average Calibration Profile" is derived by averaging of the profiles from each of their respective replicate profiles to produce an average profile.

The primary function of a Calibration Database is to provide an average optical calibration factor (OCF), which is used to calculate the number of target molecules from a sample within the Experiment Database. The Optical Calibration section provides details as to how an OCF is generated and its function in correlating fluorescence units to DNA mass.
