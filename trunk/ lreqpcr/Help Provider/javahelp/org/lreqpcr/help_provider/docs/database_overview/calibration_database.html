Calibration Database

Similar to the Experiment Database, data is stored in a tree-like structure in which calibration profiles are listed under the corresponding Reaction Setup. An "Average Calibration Profile" is derived by averaging of the profiles from each of their respective replicate profiles to produce an average profile.

The primary function of a Calibration Database is to provide an average optical calibration factor (OCF), which is used to calculate the number of target molecules within an Experiment Database. Note that a calibration database can only contain one reaction setup, so that a new calibration database must be created for each type of reaction setup.
