Getting Started

If you are unfamiliar with LRE, the LRE introduction page provides an overview that should provide a basic understanding of LRE. The three LRE overview videos available on the LRE web site (sites.google.com/site/lreqpcr) also provides a more technical description of the origins and implementation of LRE, along with describing methods that can be used to assess the dynamic range and quantitative accuracy of qPCR.

The LRE Analyzer Introduction page provides a general overview of how the program functions. It is also recommended that the demonstration database files be opened in order to provide an introduction to LRE using high quality data sets. It is further recommended that the Implementing LRE-based Assays section be consulted before attempting to apply LRE analysis to your own data sets.

The Data Import section provides instructions about how to import data from various software packages, along with the ability to manually import data.
