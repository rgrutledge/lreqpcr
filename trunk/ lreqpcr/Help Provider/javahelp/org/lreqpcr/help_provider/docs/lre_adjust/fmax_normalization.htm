Fmax Normalization

Fmax normalization is applied by selecting the "Fmax Normalize" checkbox located just above the explorer panel tree, while deselecting the checkbox removes Fmax normalization. In addition, an asterisk (*) is added to the profile label to indicate that Fmax normalization has been applied.

Fmax normalization was introduced primarily to correct for well-to-well differences in fluorescence scale generated by the instrument's optical system. Although such variances are generally small, the simple natural of Fmax normalization provides an attractive option for potentially increasing the accuracy of LRE quantification. As discussed below, analysis of a run's average Fmax also introduces an additional quality control component.

Because LRE determines target quantity expressed in fluorescence units, the quantitative accuracy of LRE is directly dependent on the optical precision of the qPCR instrument. In particular, LRE quantification assumes that all wells generate identical fluorescence signal intensities. However, as illustrated below, large differences in profile height brings this assumption into question.

Under the LRE model, quantifying profile height is an integral part of target quantification. Defined as the maximal fluorescence or Fmax, LRE analysis provides a quantitative foundation for addressing differences in fluorescence scale. This is accomplished by first determining the average Fmax across all wells within each run. This provides a reference point to which the Fmax generated by an individual well can be compared.

Also worthy of note is that the C1/2 values are nearly identical (as is the case for all the other profiles in this run), whereas the corresponding Cq values difference by 1.0 cycles. Although still under investigation, in the absence of other factors (see below), these differences in profile height appear to reflect differences in the fluorescence scale generated by the instrument's optical system.

Fmax normalization utilizes a profile's Fmax to adjust the predicted target quantity based on the relative difference to the run's average Fmax. For example, if a profile generates a Fmax that is 50% higher than the average Fmax , it will generate a 50% overestimation of target quantity. Fmax normalization corrects the bias by simply lowering the predicted target quantity by 50%. If the profile height is smaller than the average, the target quantity is similarly increased:

The average Fmax also provides a simple method for assessing the variance of fluorescence scale across all wells within a run, as indicated by the coefficient of variation (CV = standard deviation / average X100%). In addition, the average Fmax allows run-to-run differences in fluorescence scale to be assessed, by comparing their respective average Fmax values:

Examining these three lambda runs reveals that despite large differences in well-to-well Fmax values, the overall variance is relatively moderate (±11% for all three runs in this example). Furthermore, the ±1.3% run-to-run variance generated here is indicative of the high level of optical precision generated by this instrument (ABI 7500), as was also evident from the analysis described in the Optical Precision section. This suggests that well-to-well differences are not necessarily due to loss of optical precision, but are rather more likely related to calibration, that is, differences in fluorescence scale assigned to an individual well.

Although these run-to-run variances have been found to be generally small, examining the average Fmax does allow any aberrations in the optics of an individual run to be identified. This could be produced, for example, by technical errors in reaction setup or aberrations in an instrument's optical system during a run. Although such situations are expected to be rare, this approach does provide an additional quality control element.

Some important qualifications
Importantly, this approach assumes the absence of other factors impacting profile height, which is not necessarily the case. For example, differences in amplification efficiency (Emax) also impacts Fmax in a linear fashion (Rutledge and Stewart 2010).

Thus, a necessary assumption of Fmax normalization, as it is applied in this version of the LRE Analyzer, is that amplification efficiency is equal, or at least similar, for all profiles within a run.

A second assumption is that all amplicons generate similar fluorescence intensities (FU/bp), which, as based on previous work (Rutledge and Stewart 2010), appears to be generally valid. However, implementing the average Fmax has revealed exceptions to this assumption. For example, the SAND F1R1 amplicon that is included in the demonstration experiment database, for unknown reasons, consistently generates Fmax values that are about 25% lower than the average Fmax. However, it is also important to note that in addition to being small, such systemic biases are repeatable, and as such, relative differences in target quantity are not impacted.
