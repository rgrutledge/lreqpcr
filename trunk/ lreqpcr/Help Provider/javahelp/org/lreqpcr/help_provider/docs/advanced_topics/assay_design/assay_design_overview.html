Designing large-scale assays

One of the major limitations of conducting large scale qPCR is the inability to optimize cycling regime to individual primer pairs. particularly when a large number of targets need to be quantified,  it is necessary to use a single cycle regime.

One of the largest limitations for large scale qPCR applications isn't is a necessity to forgo extensive testing and optimization of individual primer sets.
