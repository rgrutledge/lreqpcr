Welcome to the LRE Analyzer Ver. 0.9

If you are unfamiliar with LRE qPCR, the LRE Overview section provides a basic description of the underlying principles. The LRE qPCR website (sites.google.com/site/lreqpcr) also provides additional information about LRE, including a three part video series describing how LRE was conceived, along with methods that were used to evaluate the accuracy and dynamic range of LRE quantification.

The Program Overview section describes how the program is structured, which, along with the demonstration databases, provides a hands-on introduction to how the program functions.

The Data Import section provides instructions on how to import data from various software packages, along with the ability to manually import data.  However, it is recommended that before attempting to analyze your own datasets Implementing LRE-based Assays should be consulted.

The Optical Calibration section describes how quantitative scale is determined, based on calibrating the fluorescence intensity of an assay. This is required for for each individual assay-instrument combination and requires that an optical calibration factor be manually entered into the experiment database window.

The Data Export section describes how data can be exported to an Excel workbook, sorted by run, sample or amplicon.

The Quality Control and Trouble Shooting sections describe various issues related to achieving and maintaining quantitative accuracy.
