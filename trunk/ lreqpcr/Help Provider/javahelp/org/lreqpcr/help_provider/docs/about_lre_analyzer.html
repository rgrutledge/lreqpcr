Welcome to the LRE Analyzer

The LRE Analyzer was developed as a standalone, platform independent desktop program that automates qPCR data analysis, which in combination with utilizing lambda gDNA as a universal quantitative standard, allows absolute quantification to be conducted with little or no user intervention. Originating from modeling PCR amplification as a sigmoidal process, the LRE Analyzer provides the automated analysis and data storage capabilities required for large-scale projects, in addition to providing quality control capabilities that are difficult, if not impossible to achieve using convention qPCR methodologies.

Note that the LRE qPCR website (sites.google.com/site/lreqpcr) provides additional information about LRE, including a three part video describing how LRE was conceived, along with a detailed overview of methods used to evaluate the accuracy and dynamic range of LRE quantification.
