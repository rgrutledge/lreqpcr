Replicate Profile Scattering

As described in the Optical Precision section, average profiles are constructed from the replicate profiles (if more than one is available) in order to increase optical precision, which can greatly improve the performance of some instruments with poor optics.

However, in order to generate a valid average profile, the replicate profiles must be reasonably clustered. As of version 0.9.3 the LRE Analyzer monitors profile clustering and will invalid an average profile if the  C1/2 values of its replicate sample profiles differ by >0.7 cycles.

Although an average of the target quantities generated by the replicate profiles are provided in the average profile label, profile scattering is generally considered to invalid the analysis unless a clear reason for the scattering can be identified.

Note that this issue is still under discussion, and modifications may be implemented in future versions of the program.
