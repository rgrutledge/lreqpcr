Emax Normalization

Emax normalization is applied by selecting the checkbox ("<100%> Emax") located just above the explorer panel tree. In addition, the LRE derived Emax displayed in the profile node label is replaced by "<100%> to signify that Emax has been fixed to 100%. Note that Emax normalization is applied to all profiles contained within the database. To return to the LRE-derived Emax , simply deselect the checkbox.

Particularly for Qiagen's QuantiTect enzyme formulation, LRE analysis has been found to generate Emax values >100%, which are most likely due to aberrant amplification kinetics that produce systematic biases in Emax determination (the source of these aberrancies remains unknown). Depending on the amplicon, Emax can reach up to 107-110%, although most amplicons generate Emax values in the 100-104% range. Empirical testing has shown that a 1% overestimation roughly generates a 10% underestimation of target quantity, although this relationship is not linear (i.e. the overestimation increases as the Emax bias becomes larger).

Extensive testing has subsequently shown that fixing Emax to 100% is effective in reducing, or in many cases eliminating, the quantitative errors caused by this systematic bias. A key component of this testing was to use "limiting dilution assay" (LDA) to provide an independent determination of target quantity, which has shown that fixing to Emax to 100% produces target quantities within 15-30% of the actual quantity, even for amplicons that generate Emax values of 107-110%. As discussed in the verification of quantitative accuracy section, LDA is simple to conduct and does not require a quantitative standard, thus providing a compelling methodology for confirmation of the quantitative accuracy of any qPCR assay, irrespective of how it is implemented

Note that fixing Emax to 100% does not modify any of the other parameters used to calculate target quantity. This is because 100% Emax is applied only during calculation of target quantity (F0), in which the Fmax is derived from the LRE analysis conducted on the profile.
