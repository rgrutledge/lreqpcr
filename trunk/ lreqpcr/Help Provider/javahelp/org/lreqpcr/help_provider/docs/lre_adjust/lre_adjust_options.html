Options for Adjusting LRE Analysis

Two options are available for adjusting how the LRE analysis is conducted.

The first addresses the propensity of Qiagen QuantiTect enzyme formulation to generate overestimations of amplification efficiency. Although the underlying mechanism is unknown, fixing Emax to 100% has been found to be effective in reducing the biases caused by this kinetic aberrancy. This section also provides an example of how limiting dilution assay (LDA) can be used to independently assess quantitative accuracy.

The second, referred to as Fmax normalization, addresses variances in instrument optics, the most prominent being well-to-well differences in the fluorescence scale. This section also provides important perspectives into how the optics of an assay impact LRE quantification.
