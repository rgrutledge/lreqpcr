Manual LRE Window Selection

Although generally not required, manual adjustment of the LRE window may be desirable. The LRE Plot within the Profile Editor window provides the ability to change the size of the LRE window by clicking on the radio buttons located on the left and right of the panel:

Note that each time the LRE window is modified, LRE analysis is repeated and all of the panels within the editor are updated. Note also that these changes are automatically saved to disk. The "Reset" button triggers an automated window selection.
