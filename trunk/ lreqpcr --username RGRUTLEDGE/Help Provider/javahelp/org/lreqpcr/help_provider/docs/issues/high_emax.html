Emax Normalization

Particularly for Qiagen's QuantiTect enzyme formulation, LRE analysis has been found to generate Emax values >100%, which are most likely due to aberrant amplification kinetics that produce systematic biases in Emax determination (the source of these aberrancies remains unknown). Depending on the amplicon, Emax can reach up to 107-110%, although most amplicons generate Emax values in the 100-104% range. Empirical testing has shown that a 1% overestimation generates about a 10% underestimation of target quantity.

Extensive testing has subsequently shown that normalizing to 100% is effective in reducing, or in many cases eliminating, the quantitative errors caused by this systematic bias. Thus, for profiles for which LRE analysis produces an Emax >100%, the LRE Analyzer calculates target quantity using an Emax = 100%.

A key component of this testing was to use "limiting dilution assay" (LDA) to provide an independent determination of target quantity, which has shown that normalization to 100% Emax produces target quantities within 15-30% of the actual quantity, even for amplicons that generate Emax values of 107-110%. As discussed in the verification of quantitative accuracy section, LDA is simple to conduct and does not require a quantitative standard, thus providing a compelling methodology for confirmation of the quantitative accuracy of any qPCR assay, irrespective of how it is implemented.
